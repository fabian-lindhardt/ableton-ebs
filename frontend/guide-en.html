Fader

Drag up/down to send continuous CC values (0-127). Perfect for volume, filters, or any continuous parameter. Click the value display to type an exact number.

Knob

Drag up/down to rotate. Great for pan, effects sends, or synth parameters. Supports "pan mode" for center-detent style.

Button / Note Pad

Click to send a MIDI note or CC burst. Ideal for triggering one-shots, samples, or scene changes.

Toggle

Click to switch between On (127) and Off (0). Perfect for mute/solo, effect bypasses, or loop toggles.

XY Pad

Drag to control two CC values simultaneously (X and Y axes). Great for Kaoss-pad style effects or filter + resonance combos.

Transport

Play, Stop, and Restart buttons for Ableton's transport controls.

üì° 5. For Broadcasters

Initial Setup

  1. Install the Local Bridge ‚Äì Download and run the local-bridge application
  2. Install Max for Live Device ‚Äì Load the M4L device in Ableton to enable track scanning
  3. Configure Extension ‚Äì Go to Extension Config in your Twitch dashboard
  4. Map MIDI Controls ‚Äì Use Ableton's MIDI Learn to map the extension controls

Configuration Panel

Access the config panel to:

  • Add custom triggers (faders, knobs, buttons, etc.)
  • Set MIDI channels and controller numbers
  • Configure Bit costs for premium controls
  • Set up VDO.Ninja room ID for audio streaming
  • Link controls to specific Ableton tracks (Smart Sync)
‚ö†Ô∏è Important: Make sure your Local Bridge is running before going live! Without it, viewer commands won't reach Ableton.

MIDI Mapping in Ableton

  1. Press Cmd/Ctrl + M to enter MIDI Map Mode
  2. Click the parameter you want to control
  3. Move the corresponding control in the extension (or use the "Learn" buttons)
  4. Press Cmd/Ctrl + M again to exit

üîß 7. Troubleshooting

"Bridge not connected"

The local bridge application isn't running or isn't reachable. Start the bridge and check your firewall settings.

Controls not responding

  • Verify MIDI mapping in Ableton (Cmd/Ctrl + M)
  • Check that the correct MIDI port is selected in the bridge
  • Ensure the VIP session is active (if controls require VIP)

No audio in VIP mode

  • Click the "üîä JOIN AUDIO" button after unlocking
  • Allow browser permissions for audio autoplay
  • Verify the VDO.Ninja room ID is correctly configured

Session Grid not updating

Make sure the Max for Live device is loaded on a track in your Ableton project. It handles track/scene metadata sync.
